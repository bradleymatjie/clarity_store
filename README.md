# clarity store


## Technologies
Project is created with:

* HTML
* SCSS (SASS)
* JAVASCRIPT
* React
* React Router
* Firebase for backend


## Run Locally
Clone the project

``` bash
  git clone https://github.com/bradleymatjie/clarity_store.git

# Go to the project directory

``` bash
  cd clarity_store

# Install dependencies

``` bash
  npm install

# Start the server

``` bash
  npm start
